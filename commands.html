COMMAND REFERENCE

Every command, grouped by bot.

NIX is three separate Discord bots. Below is every slash command, what it does, and which bot owns it. Permissions are noted where they apply.

🛠️ NIX Setup
Server templates, export/import, welcome flow, custom commands
Add Setup →
Server Building · owner only
/setupBuild a server from a template
/undoReverse the last setup run
/exportSave server layout as a shareable code
/import <code>Build server from someone's code
/exportsList codes you have created
/revoke <code>Disable an export code
Server Tools · admin only
/welcomeConfigure welcome flow + auto-role
/customcmdPer-server ! text commands (max 50)
Info
/helpCommand reference for this bot
/botinfoVersion, ping, uptime, server count
/inviteGet the invite link
🌸 NIX Anime
Anime lookup, quiz, watch parties, release feed, AniList integration
Add Anime →
Lookup · MyAnimeList
/anime <name>Anime details — cover, score, episodes, genres
/manga <name>Manga / manhwa / manhua / LN details
/character <name>Character bio + top appearances
/seiyuu <name>Voice actor + top voiced roles
/streaming <anime>Where to legally watch
/compareSide-by-side anime comparison
Random Pickers
/random animeRandom anime, no filters
/random mangaRandom manga
/random characterRandom character
/random quoteRandom anime quote
/randomanimeRandom anime with genre/status/score filters
Engagement
/quiz [mode]Anime quiz (synopsis / cover / character)
/quizleaderboardTop players in this server
/tierlistBuild S/A/B/C/D/F tier lists
/anilist link/showLink AniList account, show profile stats
/remindme add/remove/listDM you when new episodes air
/watchparty scheduleCreate Discord scheduled event
/animerolesPost genre role picker (admin)
Info
