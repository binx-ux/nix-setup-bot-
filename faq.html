Getting Started
Click the Add to Server button on the homepage or the commands page. You'll need the Manage Server permission in the target server. The bot requests Administrator permissions so it can perform moderation actions on your behalf.
No configuration is required to start. Commands like /meme, /rank, and /help work immediately. Run /setup (server owner only) if you'd like NIX Bot to automatically build out your server's channel structure using a template.
Moderation commands like /server ban, /server kick, and /channel lock require specific elevated permissions. Requesting Administrator simplifies setup so you don't have to manually assign a dozen individual permissions. You can reduce permissions after setup if you prefer — the bot will only fail the commands that need permissions you've removed.
Discord can take up to one hour to propagate global slash commands after the bot first joins or restarts. Try restarting your Discord client first. If commands still don't appear after an hour, make sure the bot has the Use Application Commands permission in the channel you're trying to use it in.
Security & Malware Scanning
Every message attachment with a dangerous extension (.exe, .dll, .bat, .cmd, .ps1, .vbs, .msi, .scr, .hta, .wsf, .reg, .lnk) is automatically downloaded and submitted to VirusTotal for analysis. Files flagged as malicious are deleted and a threat report is posted. Suspicious files trigger a warning. Clean files get a ✅ reaction.
The bot owner needs to set a VT_API_KEY environment variable for the auto-scanning to function. Free VirusTotal API keys are available at virustotal.com — the free tier is sufficient for most servers. Without the key, auto-scanning is silently skipped and the /scan command won't work.

Commands & Permissions
Moderation commands (/server kick, /server ban, /server timeout, etc.) require the user to have the relevant Discord permission (e.g. Ban Members to ban). Channel commands require Manage Channels. The bot checks permissions before executing — it won't let someone without the right role run destructive commands.
Reddit rate-limits bot traffic. The 8-second per-user cooldown prevents a single user from hammering the Reddit API and causing it to fail for everyone else in the server. The cooldown is per-user, not per-server — multiple people can use /meme simultaneously without triggering it.

Music
The music system is powered by Lavalink and supports YouTube, Spotify, SoundCloud, and direct audio URLs. Paste a song name, YouTube link, Spotify track/playlist, or SoundCloud URL into /play and it will queue up automatically.
Music commands require the DJ role or higher (Mod, Admin, Owner). You can assign the DJ role to any role in your server — anyone with it can queue, skip, pause, and control the player.
Privacy & Data
NIX Bot stores XP and level data per user per server (for the leveling system) and the blacklist (user IDs only). No message content is logged, no personal data is sold or shared, and no analytics are collected. File scan results are not stored — only the VirusTotal API response is used in the moment and discarded.
