Features
When someone uploads a file with a potentially dangerous extension (such as .exe, .dll, .bat, or .ps1), NIX automatically scans it using VirusTotal. If the file is flagged as malicious it is deleted and a warning is posted. If it is suspicious, the channel gets a heads-up. Clean files get a checkmark reaction. This requires the bot to stay in the server and a VirusTotal API key to be configured.
The Code/AI Community template includes a live AI news feed. NIX stays in the server and periodically posts the latest AI industry news to designated channels like #ai-news and #model-releases. Because it needs to stay active, the Build and Leave option is disabled for this template.
