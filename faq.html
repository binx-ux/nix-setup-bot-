The Bot Family
NIX is three separate Discord bots that share a website and changelog:

🛠️ NIX Setup — server templates, /export, /import, welcome flow, custom commands
🌸 NIX Anime — anime lookup, quiz, watch parties, episode reminders, release feed
📊 NIX Tools — website uptime monitoring, file scanning, /avatar, /banner, /afk, AI news

Each one focuses on one job so admins can invite only what they need. Pick the bots you want here.
No. They work independently. Most servers want all three because they don't overlap, but if you're only here for /setup you can skip the other two. If your server is anime-themed but you don't care about website monitoring, invite Setup + Anime and skip Tools. Mix and match however you want.
Four reasons: focused permissions (admins grant each bot only what it needs), failure isolation (one bot crashing doesn't take down the others), cleaner help (each bot's /help shows just its own commands), and install what you want (don't need monitoring? skip NIX Tools).

Export & Import
/export packages your server's channel layout (categories and channels) into a short shareable code like NIX-A3K7B2. The code gets DMed to you privately. Anyone you share it with can run /import to build the same channel structure on their own server.
Only the channel layout: category names and channel names (text + voice). Roles are NOT exported by design — this prevents anyone from sharing roles with malicious permissions. Also never included: messages, member lists, server settings, audit logs, integrations, custom emojis, or anything user-specific. Channels-only by design.
Every export goes through a 4-layer moderation pipeline before a code is generated: a hard-block keyword list (scams, phishing, malware tools), pattern detection for homoglyph attacks and obfuscation, size limits, and AI moderation via OpenAI's content policy API. Exports get re-checked again on import in case rules tightened. Anything flagged is rejected with no code created and the rejection is logged for review.
