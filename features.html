FEATURES

Everything the NIX family does.

Three Discord bots, no overlap. Each one focuses on a single domain and does it well. All features below are free forever.

🛠️

Server Building

NIX Setup
⚡ 49+ Server Templates

Pick from professionally designed templates spanning gaming, anime, roleplay, content creator, streaming, dev, and more. Run /setup and your entire server structure is built in under a minute.

📦 Export / Import

Package your server\'s layout into a short shareable code with /export. Anyone can /import it to rebuild the same structure. Channels-only by design — roles never leave your server.

🛡️ 4-Layer Moderation

Every exported template runs through hard-block keywords, pattern detection, size limits, and the OpenAI moderation API before a code is generated. Nothing harmful ships.

👋 Welcome Flow

/welcome sets up a fully configurable welcome message with token interpolation ({user}, {server}, {membercount}), auto-role assignment, and a live preview via /welcome test.

⚙️ Custom Commands

Admins create per-server ! text commands without leaving Discord. Up to 50 per server, 1500-char responses. Members get instant replies on demand.

↩️ One-Click Undo

Changed your mind? Run /undo to reverse the last setup. Live progress shown — see what\'s being deleted in real time.

🌸

Anime Tools

NIX Anime
🔎 MyAnimeList Lookup

/anime, /manga, /character, /seiyuu — instant lookups with cover art, score, episodes, genres, voice actors, and bio. Cached for speed.

🎬 Auto Release Feed

Bot polls Jikan every 15 minutes and auto-posts new episode drops to any channel named #new-episodes. Includes cover, MAL link, and auto-spawns a 24h discussion thread.

🔔 Episode Reminders

/remindme add <anime> — get a DM when a new episode airs. Wired into the same release feed, so reminders fire the moment Jikan reports the drop.

🎌 Anime Quiz

Three modes: synopsis, cover art, character. 25-second timer, 4 multiple choice. Tracks correct/wrong/streak per user. Public leaderboard on the website.

📊 AniList Integration

Link your AniList account with /anilist link, then /anilist show displays your full profile — anime/manga counts, watch time, mean scores, top favorites.

📺 Watch Parties

/watchparty schedule creates a Discord scheduled event with the anime\'s cover as the event banner. Members can RSVP and get notified before showtime.

📅 Daily Digest

Auto-posts the top 5 trending airing anime to #anime-general every day at 12:00 UTC. Spotlight section highlights the #1 with full synopsis.

🎭 Role Picker

/animeroles auto-posts a one-click genre role picker — Shounen, Isekai, Romance, Mecha, Horror, Sub/Dub preference. Roles created automatically.

📊

Tools & Monitoring

NIX Tools
🌐 Website Uptime Monitoring

/monitor add <url> — track any URL\'s uptime per server. Bot maintains a single status embed per channel, pings on state transitions (down OR recovery), and dedupes URL checks across guilds.

🦠 VirusTotal File Scanning

Auto-scans uploaded executables, DLLs, scripts, and installers through 70+ engines. Malicious files are auto-deleted with a warning. Manual scans via /scan.

📰 AI News Feed

Polls RSS feeds from 12 AI labs (OpenAI, Anthropic, Google AI, DeepMind, Hugging Face, Mistral, xAI, Cohere, Stability, Black Forest Labs, Runway, Midjourney) every 30 min and posts to dedicated channels.

💤 AFK System

/afk <reason> — bot replies on your behalf if someone pings you. Auto-clears when you next speak.

🖼️ Avatar / Banner

/avatar [user] and /banner user/server fetch high-resolution avatars and banners with one click.

🟢 Status Channel Rename

Designated status channel auto-renames between 🟢-online, 🟡-updating, and 🔴-offline based on bot lifecycle. Visible at a glance.

ℹ️ Server / User Info

/serverinfo shows full server stats. /ping, /uptime for bot health checks. /meme for random Reddit memes.
