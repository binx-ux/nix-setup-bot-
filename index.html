Deep Dive

Malware Shield,
explained.

NIX Bot monitors every message attachment in real time. When a dangerous file type is posted, it's automatically downloaded, hashed, and sent to VirusTotal for analysis before anyone can open it.

  • Scans .exe, .dll, .bat, .cmd, .ps1, .vbs, .msi, .scr, .hta, .wsf, .reg, .lnk automatically
  • Files flagged as malicious are deleted and a full threat report is posted in the channel
  • Suspicious-but-not-confirmed files trigger a warning reply so users can decide
  • Requires a free VirusTotal API key — set once, works everywhere
