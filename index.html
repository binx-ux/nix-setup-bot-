HOW IT WORKS

Four steps. No configuration.

From empty server to fully built community in under a minute. Here's exactly what happens at each step.

01

Invite the bot

Add NIX to your Discord server with a single click. It requests only the four permissions it needs: Manage Channels, Manage Roles, View Channels, and Send Messages. Nothing else.

  • Takes about 10 seconds end to end
  • You can kick the bot at any time with no side effects
  • No data is collected about your existing members or messages
Add to Server →
02

Run /setup and pick a template

Type /setup in any channel. Only the server owner can run it. A category dropdown appears first (Shooter, Survival, VR, Trading, plus 13 more), then a template picker, then a full preview.

  • 47+ templates across 17 picker categories
  • See every role and channel before you confirm anything
  • "Build & Leave" or "Build & Stay" — your choice
  • Cancel any time before the final confirm
03

Server is built in real time

NIX creates every role bottom-up so the hierarchy comes out right, then builds categories and channels with the correct permission tiers. You watch live progress in the embed as it runs.

  • Live counter shows roles and channels as they're created
  • Existing roles or channels with the same name are skipped, not overwritten
  • Assign Admin and Moderator roles to your team without leaving Discord
  • Full setup summary delivered to your DMs — or in-channel if DMs are off
  • Run /undo to roll everything back if you change your mind
04

Share or duplicate your server

Once your server is set up the way you like it, run /export to package the channel layout into a shareable code like NIX-A3K7B2. Send the code to a friend, they run /import and rebuild your structure on their own server in seconds.

  • Channels-only by design — no roles travel, no security risk
  • Every export passes 4 moderation layers before a code is generated
  • Codes auto-expire after 30 days, revoke anytime with /revoke
  • Track all your codes with /exports
BEHIND THE SCENES

Things NIX handles automatically

🛡️
File scanning

Every executable, script, or installer uploaded to your server is auto-scanned with VirusTotal against 70+ engines. Malicious files are deleted automatically.

📰
AI news feeds

12 AI labs polled every 30 minutes. Each source posts to its own dedicated channel if you have one (#openai, #anthropic, #midjourney, etc.).

🚫
Scam protection on exports

Hard-block keywords, homoglyph attack detection, size limits, and OpenAI's Moderation API screen every export before a code is generated.

📊
Setup history tracking

Every /setup is logged with role and channel IDs so /undo always knows exactly what to remove.
