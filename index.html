HOW IT WORKS

Three steps. No configuration.

01

Invite the bot

Add NIX to your Discord server with a single click. It requests only the permissions needed to build your server structure.

Add to Server →
02

Run /setup

Type /setup in any channel. A dropdown appears with all 25+ templates. Select the one that fits your community.

03

Done — it leaves

NIX creates every role, channel, and category instantly. You receive a DM with a full summary and optional invite links for Carl-bot and Arcane. NIX can leave your server automatically so it stays clean.

FEATURES

Everything handled automatically.

Instant Roles

Every template comes with a curated set of roles — admin, moderator, member, bots — configured with the right colors and hierarchy out of the box.

Smart Channels

Channels and categories tailored to each community type. The Code/AI template even includes a live AI news feed channel.

Permission Tiers

Channel permissions are wired up correctly from the start. No manual permission editing — everything just works the moment setup completes.

DM Summary

After setup, NIX sends the server owner a direct message with a full breakdown of what was created and recommended next steps.

Build and Leave

NIX can leave your server automatically once setup is complete. No lingering bot presence — a clean server from day one.

Bot Invite Links

Your DM summary includes ready-to-use invite links for Carl-bot and Arcane, so you can add the most popular moderation and leveling bots in one click.
